cpu_ray_tracer:#

This project models light using common ray tracing methods, and some path tracing for indirect lighting.

This project was made in rust, and after this i intend to make another ray tracer built using GPU computing.

Making the ray tracer was really fun, as building up, the images generated slowly got more and more realistic, and it was really nice to show friends and family.

The goal of this project was to learn more about ray tracing as a method of 3D rendering, I intend to do a Rasterizer in the future.

chip_8_emulator:#

This project emulates games made for the chip 8 emulator

Getting the final result was amazing, but i will leave warning that it doesn't work with a minorty of chip 8 software, im pretty sure this is because of how certain games think instructions are handles differently, so if i make one game work, another wont

I'm a massive fan of retro stuff, so this project was pretty much perfect for me, even if it is the easiest project in the emulation, however i intend to move on to emulating the NES, so that's quite the step

Prior to making this i had a streak of coding in assembly (x86 to be exact), and i think a project like this has majorly developed my understanding of how CPU's work, as encoding my own instruction set is a pretty good way to gain experience
