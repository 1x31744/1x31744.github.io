cpu_ray_tracer:#

This project models light using common ray tracing methods, and some path tracing for indirect lighting.

This project was made in rust, and after this i intend to make another ray tracer built using GPU computing.

Making the ray tracer was really fun, as building up, the images generated slowly got more and more realistic, and it was really nice to show friends and family.

The goal of this project was to learn more about ray tracing as a method of 3D rendering, I intend to do a Rasterizer in the future.
